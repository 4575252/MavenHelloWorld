# 关于本项目
本项目是一个简易的hello world欢迎示例，基于maven技术


### 使用手册

* 访问入口: http://[ip:port]/hello，ip、端口以实际环境为准。
* 运行方式: 将hello.war丢到tomcat中的webapp中


### 技术参考
> maven中使用的骨架是`org.apache.maven.archetypes:maven-archetype-webapp`
